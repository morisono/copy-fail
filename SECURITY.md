# Security Policy

## Authorized Use

This project is a proof of concept for security research and controlled lab validation. Use it only on systems you own or where you have explicit authorization.

Do not use this project against third-party systems, production assets, or environments where you do not have permission to perform security testing.

## Reporting Issues

If you find a bug in this repository, report it through the project's issue tracker or through a private maintainer contact if one is available.

Avoid posting sensitive system details, private logs, credentials, or information that could expose third-party systems.

## Supported Versions

| Version | Supported |
| --- | --- |
| `0.1.x` | Yes |

## Scope

Security reports should focus on the repository code, documentation, and packaging. Reports about unauthorized use, third-party targets, or requests to adapt the PoC for real-world intrusion are out of scope.
